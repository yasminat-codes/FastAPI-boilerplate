## [0.1.0] - 2026-04-10

### Added

- App factory pattern with FastAPI application setup
- JWT authentication with token generation and validation
- Webhook ingestion framework for handling third-party events
- Workflow orchestration with task composition and state management
- Background jobs with ARQ queue system
- Observability stack with Prometheus metrics, OpenTelemetry tracing, and Sentry error tracking
- Docker deployment assets with multi-stage builds and security hardening
- MkDocs documentation site with comprehensive guides
- CI/CD pipeline with 9 GitHub Actions workflow files for testing, linting, and deployment
